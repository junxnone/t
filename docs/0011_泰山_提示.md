# 友情提示
- 携带身份证
- 提前购买门票预约日期
- 关注天气及索道停运信息
- 山顶物价贵，考虑体力的情况下，携带部分食物和饮料
- 山上温度低，看日出时温度更低，根据温度携带厚的保暖衣物
- 节假日人多，索道排队，错峰出行
- 夜爬携带手电筒，不宜过早，否则等待时间较长
- 禁止携带火机火种及帐篷等露营设施
- 禁止携带无人机
- 未使用门票，预约日期后三天内可退款
